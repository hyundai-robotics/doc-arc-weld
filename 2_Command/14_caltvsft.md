# 2.14 CalTVSft

- 설명 
  
    CalTVSft 문은 입력된 두 포즈변수에서 툴을 세울 수 있는 시프트값을 구하는 명령문 입니다.

- 문법
  
    - CalTVSft <포즈1>,<포즈2>,<시프트 변수>

- 파라미터
  
   ① 포즈1
     - 내용 : 센싱에 의해 구해진 포즈1
   
   ② 포즈2
     - 내용 : 센싱에 의해 구해진 포즈2

   ③ 시프트변수
     - 내용 : 두 포즈값에 의해 구해진 툴의 기울기를 똑바로 세우기 위한 시프트 변수
      
</br>  

- 사용 예
  
   - ```CalTVSft LP1,LP2,LR1```
